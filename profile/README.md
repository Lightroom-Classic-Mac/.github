## Overview

<a href="#">Lightroom Classic for Mac</a> is Adobe's desktop-anchored photo management and editing application, purpose-built for photographers who work with large image libraries and require a consistent, structured workflow from import to delivery. Unlike cloud-first alternatives, Lightroom Classic stores its catalog and original files locally, giving photographers full control over where their images live and how they are organized — a critical consideration for professionals managing terabytes of client work.

The <a href="#">catalog system</a> is the foundation of Lightroom Classic's organizational power. Every image imported into Lightroom Classic is indexed in a SQLite database that records metadata, editing history, virtual copies, and collection membership. <a href="#">Smart collections</a> group photos automatically based on rules — camera body, lens, focal length, rating, color label, date range, or any combination — updating dynamically as new images meet the criteria. Keywording, star ratings, flags, and color labels layer additional organizational dimensions that make filtering thousands of images to a specific set fast and predictable.

<a href="#">Non-destructive RAW processing</a> is Lightroom Classic's editing model. Every adjustment — exposure, white balance, tone curve, color grading, sharpening, noise reduction — is stored as a set of instructions rather than applied to the original file. The RAW file remains untouched at all times. <a href="#">Develop module tools</a> provide precise control over every tonal and color parameter: the Basic panel handles global exposure and white balance, HSL controls adjust individual color ranges, the Tone Curve shapes contrast with point or parametric control, and the Color Grading panel applies split toning to shadows, midtones, and highlights independently.

The <a href="#">masking system</a> introduced in recent versions uses Adobe Sensei machine learning to create precise selections with minimal manual work. Subject Select, Sky Select, and Background Select generate accurate masks from a single click. <a href="#">People masks</a> detect faces and separate body regions — skin, hair, eyes, lips, teeth, eyebrows — for targeted portrait retouching without manual brushwork. These AI-powered selections can be refined with brush additions or subtractions and combined using intersection and exclusion logic for complex, layered masking workflows.

## Key Features

- <a href="#">Local catalog system</a> indexes millions of images with smart collections keywords ratings and color labels for instant filtering
- <a href="#">Non-destructive RAW processing</a> preserves original files while storing all edits as reversible instructions
- <a href="#">AI Subject Sky and People masking</a> generates precise selections from a single click using Adobe Sensei machine learning
- <a href="#">Tone curve HSL and Color Grading</a> panels deliver granular control over contrast color balance and split toning
- <a href="#">Batch editing with presets</a> applies consistent processing to hundreds of images simultaneously with one-click synchronization
- <a href="#">Lens correction and geometry</a> tools fix distortion chromatic aberration and perspective with automatic profile matching
- <a href="#">Tethered shooting support</a> imports images directly from connected cameras into the catalog during live shoots
- <a href="#">Flexible export engine</a> outputs JPEG TIFF PNG DNG and PSD with custom sizing sharpening metadata and watermarking
- <a href="#">Plugin ecosystem</a> connects Lightroom Classic to external editors print labs and publishing platforms via third-party extensions

## Additional Information

Lightroom Classic's develop history panel records every adjustment made to every image, making it possible to step backward to any point in an image's editing history or compare the current state with any previous version. Virtual copies allow multiple edits of the same RAW file to coexist in the catalog without duplicating the original, enabling side-by-side comparison of different processing approaches.

Adobe updates Lightroom Classic regularly with improvements to the AI masking tools, new camera and lens profiles, and performance optimizations for Apple Silicon. The application is tightly integrated with Adobe Camera Raw, sharing the same processing engine and ensuring that edits translate accurately when images move between Lightroom Classic, Photoshop, and other Creative Cloud applications.
